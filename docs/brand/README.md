# kx brand assets

The kx mark combines a command path (`k`) with a multi-context junction (`x`).
The three green nodes represent selected Kubernetes contexts.

| Asset | Use |
| --- | --- |
| `kx-icon.svg` / `kx-icon.png` | GitHub avatar, Krew, favicon, small square placements |
| `kx-logo.svg` / `kx-logo.png` | README, documentation, light backgrounds |
| `kx-icon-mono.svg` | Single-color printing, stickers, engraving |
| `kx-social-preview.svg` / `kx-social-preview.png` | GitHub social preview and launch posts |

Primary colors:

- background: `#0d1117`
- command path: `#58a6ff` to `#388bfd`
- context nodes: `#7ee787`
- foreground: `#f0f6fc`

Keep the icon square, preserve its rounded background, and do not remove the
three context nodes at small sizes.
